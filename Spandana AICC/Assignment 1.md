# Artificial Intelligence (AI)

Definition:
AI is the simulation of human intelligence in machines.

Key Areas:
- Problem solving
- Reasoning
- Planning
- Natural Language Processing
- Computer Vision

Types:
1. Narrow AI
2. General AI
3. Super AI

Approaches:
- Rule-based systems
- Search algorithms
- Machine Learning

Examples:
- Chatbots
- Self-driving cars
- Voice assistants


# Machine Learning

Definition:
ML is a subset of AI that allows systems to learn from data.

Workflow:
1. Data collection
2. Data cleaning
3. Model training
4. Evaluation
5. Prediction

Key Concepts:
- Features
- Labels
- Training vs Testing
- Overfitting / Underfitting

# Deep Learning

Definition:
Subset of ML using neural networks.

Key Features:
- Works with large data
- Automatic feature extraction
- High accuracy in complex tasks
